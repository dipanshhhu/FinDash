💰 FinDash - Finance Dashboard

📌 Overview

FinDash is a modern finance dashboard that helps track income, expenses, and balance with interactive UI.

🚀 Features

* Dashboard summary (Balance, Income, Expense)
* Interactive donut chart
* Transactions table
* Search, filter, and sorting
* Role-based UI (Admin / Viewer)
* Insights section

🛠 Tech Used

* HTML
* CSS
* JavaScript
* Chart.js
